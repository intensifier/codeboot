Dites bonjour à codeBoot

codeBoot est un environnement intégré de programmation conçu pour l'enseignement



Essayez le!

Fonctionnalités

Optimisez à la fois l'apprentissage et l'enseignement

Python et JavaScript

codeBoot contient des interprètes Python et JavaScript écrits en JavaScript, exécutés dans l'environnement de votre fureteur, qui ne nécessitent aucune installation.

Interprétation pas-à-pas

codeBoot vous permet d'exécuter le code Python ou JavaScript pas-à-pas, favorisant un meilleur apprentissage pour vos élèves.

Programmation web

codeBoot intègre des fonctionnalités de programmation web à son éditeur.

Graphiques

Produisez des graphes à partir de données diverses. codeBoot s'intègre à la librairie Vega-lite et expose un API simplifié.

Classeur intégré

codeBoot vous permet d'éditer des fichiers CSV en tant que classeurs afin de faciliter l'échange et la consommation de données.

Interface avec des librairies tierces

codeBoot vous permet d'interfacer avec des librairies JavaScript tierces, que vous pouvez inclure dans vos applications en utilisant son FFI.
